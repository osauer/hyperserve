About This Example

Static File Serving

This example demonstrates how HyperServe can serve static files like HTML, CSS, and JavaScript.

Features:

  • Automatic index.html: Requests to / serve index.html
  • Directory structure: Files are organized in subdirectories
  • Security headers: Proper headers for static content
  • Mixed routes: Static files + custom API endpoints

Server Code:

// Serve static files
server.HandleStatic("/")

// Add custom API routes
server.HandleFunc("/api/status", handler)

That's all you need to serve static files!
